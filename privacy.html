Personal Information

When you use SwiftShopr, we collect the following personal information:

  • Email addresses for account creation and login
  • User IDs generated by our authentication system
  • Device identifiers for app functionality and security
  • Purchase history including items scanned, purchased, and digital receipts
  • Product interaction data such as barcode scans and cart activity

Technical Information

We also collect technical data to improve your experience:

  • Performance data including app launch times and response speeds
  • Crash reports (anonymized) to improve app stability
  • Usage analytics to understand how you interact with the app
Note: We collect only the minimum information necessary to provide our services. We do not collect unnecessary personal data.

üõ°Ô∏è Data Security

We take data security seriously and implement industry-standard measures to protect your information:

  • Encryption - All data in transit is encrypted using SSL/TLS protocols
  • Secure storage - Data at rest is encrypted and stored securely
  • Access controls - Only authorized personnel have access to personal data
  • Regular audits - We conduct security assessments to identify and address vulnerabilities

Payment Information: Payment information is processed securely and is never stored on our servers. When using card payments, transactions are processed through certified payment processors. When using USDC, transactions are settled directly on the blockchain without storing sensitive data.

Note: While we implement strong security measures, no system is 100% secure. We encourage you to use strong passwords and protect your account credentials.
